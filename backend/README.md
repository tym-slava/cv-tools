# CV Tools Backend

Backend API для CV Tools додатку.

## Установка

```bash
npm install
```

## Запуск

### Режим розробки
```bash
npm run dev
```

### Продакшн
```bash
npm start
```

## API Endpoints

- `GET /` - Інформація про API
- `GET /api/health` - Перевірка стану сервера

## Змінні середовища

Створіть файл `.env` з наступними змінними:

```
PORT=5000
NODE_ENV=development
```
